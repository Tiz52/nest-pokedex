# Execute in development mode

1. Clone the repository
2. Execute the following command:
```
yarn install
```
3. U need to has NestJS installed
```
npm i -g @nestjs/cli
```
4. Lift the database
```
docker-compose up -d
```

## Stack overview
* MongoDB
* NestJS
* TypeScript
* Docker
